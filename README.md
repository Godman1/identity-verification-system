This is a Spring Boot backend application for verifying users based on their NIN (National Identification Number), built for community identity and payment verification.

 Features

- User Registration with NIN and Phone Number
- Mock NIN Verification Logic
- Admin Authentication with JWT
- Admin Dashboard: Approve / Reject / View / Delete Users
- Timestamps for auditing
- Export Verified Users to CSV
- API secured with CORS support

Technologies Used

- Java 21
- Spring Boot
- Spring Security (JWT Authentication)
- Lombok
- MySQL
- Maven
- Swagger UI for API testing
